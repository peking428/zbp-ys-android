## 🚀 完整操作步骤

### 第一步：重启电脑

Git安装后需要重启电脑才能生效。

### 第二步：打开PowerShell，执行以下命令

```powershell
# 进入项目目录
cd "C:\Users\peking428\Desktop\工作 -szfh\AFDX\GLM-5\android_app"

# 初始化Git仓库
git init
git branch -M main

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: ZBP压缩解压工具 v3.0"
```

### 第三步：在GitHub创建仓库

1. 打开浏览器访问：https://github.com/new
2. 仓库名称填写：`zbp-ys-android`
3. 选择 **Public** 或 **Private**
4. **不要**勾选 "Add a README file"
5. 点击 **Create repository**

### 第四步：推送代码到GitHub

```powershell
# 替换 YOUR_USERNAME 为你的GitHub用户名
git remote add origin https://github.com/YOUR_USERNAME/zbp-ys-android.git
git push -u origin main
```

### 第五步：等待自动构建

1. 推送成功后，GitHub Actions会自动开始构建
2. 访问：https://github.com/YOUR_USERNAME/zbp-ys-android/actions
3. 等待构建完成（约15-20分钟）

### 第六步：下载APK

1. 构建完成后，点击最新的workflow
2. 在页面底部的 **Artifacts** 中找到 `zbp_ys_apk`
3. 点击下载，解压得到APK文件

---

## 📱 安装到手机

1. 将APK传输到手机
2. 打开APK文件
3. 允许安装未知来源应用
4. 完成安装

## ❓ 常见问题

**Q: 推送时提示需要登录？**
A: 使用GitHub CLI登录：`gh auth login` 或使用Personal Access Token

**Q: 构建失败？**
A: 检查Actions日志，通常是网络问题，重新运行即可

**Q: 鸿蒙4.3能用吗？**
A: 鸿蒙4.3（纯血鸿蒙）不支持Android应用
